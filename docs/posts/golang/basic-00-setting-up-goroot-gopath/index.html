[Basic - 00] Setting up GOROOT, GOPATH

2020-09-11
golang
golang

Open ~/.bashrc file with vim to edit:

vim ~/.bashrc

Add following lines

export GOROOT=/usr/local/go
export PATH=$PATH:$GOROOT/bin

# Wherever, up to you
export GOPATH=/root/codekeeper/go
export PATH=$PATH:$GOPATH/bin
