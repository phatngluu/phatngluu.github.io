Set time and timezone

2020-10-06
linux
linux 
sudo apt install policykit-1

sudo apt-get install ntp

timedatectl set-ntp false

timedatectl set-timezone Asia/Ho_Chi_Minh

timedatectl set-ntp false
Or run
sudo date -s "Tuesday, October 6, 2020 6:25:09 PM GMT+07:00"
